<product_requirements>
My Host IQ is a SaaS platform for property hosts to provide an AI assistant to their guests.

**Initial State:** The platform allowed hosts to manage properties, which would feed data to an AI chatbot. It included features like user authentication (for hosts), whitelabeling, iCal calendar synchronization to automatically create guest bookings from Airbnb/Booking.com, and a secure guest login system where guests authenticated with their name and booking dates to access the chat.

**Pivoted & Current State:** The core product has been significantly simplified due to limitations with iCal data. The requirement for guest login and all related functionality (iCal sync, guest bookings, automatic emails) has been **completely removed**.

The current product provides each apartment with a unique, unlisted URL to a public chatbot. Anyone with the link can access the chatbot without any login or verification. Access is rate-limited to 100 messages per apartment per day. Hosts are now expected to manually share this link with guests via their booking platform's messaging system. A new tutorial section has been added to the host dashboard to guide them through this manual process for Airbnb and Booking.com. The AI has been enhanced to provide Google Maps links for its location-based recommendations.
</product_requirements>

<key_technical_concepts>
- **Full-stack:** React.js (with Shadcn/UI, Tailwind CSS) frontend and a FastAPI (Python) backend.
- **Database:** MongoDB, using UUIDs for document IDs.
- **AI/LLM:** System prompt engineering to provide context and behavioral guidelines to an Emergent LLM. The prompt was updated to include instructions for generating Google Maps links.
- **Authentication:** JWT-based for hosts/admins. All guest authentication has been removed.
- **Rate Limiting:** A new per-apartment, per-day rate limit (100 queries/day) was implemented on the chatbot endpoint.
- **Code Refactoring:** Major removal of features (iCal, Guest Login, APScheduler) from both frontend and backend codebases.
</key_technical_concepts>
