<product_requirements>
My Host IQ is a SaaS platform enabling apartment/hotel owners (Hosts) to provide guests with a personalized AI assistant. Hosts register, add apartment details, house rules, and local recommendations, generating a shareable link for guests. Guests use this link to chat with the AI assistant for instant answers. The platform also includes Admin roles for managing hosts and subscriptions.

Initial MVP focused on:
- AI chat interface for guests leveraging host-provided JSON data and predefined prompt templates.
- Basic host dashboard to add apartment info and recommendations.

Subsequent user requests expanded the scope significantly:
- **Authentication**: Register and sign-in functionality.
- **Whitelabeling**: Custom branding (logo, colors, fonts, backgrounds, button styles), personalized URLs/subdomains.
- **Analytics Dashboard**: Guest usage, most frequent questions, AI response tracking (Q&A pairs), performance insights, peak usage.
- **iCal Integration**: Link iCal for booking sync; AI to parse email/phone from iCal events and automatically send WhatsApp/email messages with the apartment's AI chat link.
- **Payment Simulation**: Integrate a payment flow simulation.
- **UI/UX Enhancement**: Redesigned landing page with a stronger hook, professional, modern, and clean UI, improved navigation, and onboarding explanations.
- **QR Code Generation**: Generate QR codes for each apartment's chat link, with PDF download option.
- **AI Agent Behavior**: Strict adherence to apartment/local recommendations, fallback logic for out-of-domain questions (notify owner or provide standard contact), configurable AI tone (relaxed, professional, neutral).
- **Host Workflow**: Ability to edit apartments after creation.
- **Security**: Email/phone validation during registration, custom domain support.
</product_requirements>

<key_technical_concepts>
- **Full-stack**: React frontend (Shadcn/UI, Tailwind CSS), FastAPI backend, MongoDB database.
- **LLM Integration**: Emergent LLM Universal Key for OpenAI/Anthropic/Google models.
- **Data Handling**: UUIDs for MongoDB IDs (not ObjectIDs), Pydantic models for data validation, ISO format for datetime.
- **Authentication**: User registration and login.
- **iCal Parsing**: Processing iCal events for booking details.
- **Notifications**: Automated WhatsApp/Email integration.
- **UI/UX**: Responsive design, custom components, branding, analytics visualization.
</key_technical_concepts>
