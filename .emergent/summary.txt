<product_requirements>
My Host IQ is a SaaS platform for apartment/hotel owners (Hosts) to provide guests with a personalized AI assistant. Hosts manage apartment details, house rules, and recommendations, generating a shareable AI chat link. The platform also supports Admin roles. Initial MVP included AI chat and basic host dashboard. User requests expanded to: full authentication, whitelabeling, comprehensive analytics, iCal integration (to parse bookings and trigger WhatsApp/email notifications with chat links), payment simulation, enhanced UI/UX, QR code generation with PDF download, strict AI agent behavior (travel-related only), apartment editing, and security (email/phone validation, custom domains). Further requests included: real (not simulated) email sending from host's email, OpenStreetMap address autocomplete, removal of Made with Emergent watermark, dynamic (non-hardcoded) analytics with meaningful insights, a functional Airbnb/Booking.com property import from URL, a clean chat UI without address display, and a full admin panel with CRUD operations for apartments and users, including secure access and platform statistics.
</product_requirements>
